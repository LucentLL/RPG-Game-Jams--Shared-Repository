# Music

Drop background-music tracks here (`.ogg` preferred for web + Android; add `.m4a`/`.mp3`
fallbacks if you hit codec issues in a specific wrapper).

Reference them in code via `AUDIO_MUSIC_BASE` from [`src/config/assets.js`](../../../../src/config/assets.js),
e.g. `AUDIO_MUSIC_BASE + 'title-theme.ogg'`.

Suggested tracks for the guild loop: title theme, guild-hall (management), quest-dispatch,
battle, victory, defeat.
