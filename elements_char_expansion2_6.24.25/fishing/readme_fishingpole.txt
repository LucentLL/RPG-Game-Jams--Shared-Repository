
The fishing pole is larger than the regular animation frames, so it's arranged slightly differently.

The intent is that you can use it as a separate object in your game, and overlay it above the character animations.

It includes three main animations, in the usual four directions:

+ The casting animation is 5-frames, and overlays onto the attack/tool-use animation

+ The single-frame idle animation works with the bow-nock animation

+ The reeling animation is a 2-frame loop that looks best overlaid on the idle post of the bow-ready animation

I've included a preview-guide sheet to help you visualize the arrangement of these animations, it has blocks of color to show the distinct frame sets. 
The sprite sheets are arranged the same way as the preview guide, but without the extra whitespaces between the animations.

I've also included a version of the fishingpole sheet with a character base, to serve as an alignment example.

I hope that this is clear enough and useful for your project. 
Thanks!

Note for RPG Maker: 
The RPGMaker folders are arranged in the same way as above and not as character sheets; it's only divided into those folders because of the difference in size. 

Because there could be different ways to implement this depending on your game's needs, I've left the arrangement the same as the preview guide, so that you can best modify or re-arrange the frames in a way that works best for you.

The bobber animation, however, is arranged in a regular character-sheet format for use in RM, because the frames are the size of a regular single-tile.
