Thanks for purchasing my asset pack.

This is the TIME ELEMENTS: CHARACTER EXPANSION 1 pack. 

This collection of sprite assets was designed as a followup to expand 
on the pieces originally included in the TIME ELEMENTS: CHARACTER CORE 
asset pack.

---======---

If you are using the Elements Generator:
You can copy and paste all of the folder in the "assets" folder directly 
into the generator directory. For additional information on using the 
generator, see the PDF file in the "extras > guide" folder.

Note: There are some assets in the "core" folder that are copied from
the original ELEMENTS CHARACTER CORE. These are some basic assets (such 
as unclothed template parts).

If you already have the core set, then these will be unnecessary.

~~ ~ ~ ~ ~ ~ ~

REMINDER FOR RPGMAKER USERS: If your ELEMENTS characters are misaligned in 
RPG Maker, use the included plugin ("extras > rmfix_plugin") to fix it.

~~ ~ ~ ~ ~ ~ ~

The "extras" folder also includes some additional files that were made
for the original CHARACTER CORE: the PDF guide, color palettes, and the
alignment-fix plugin for RPG Maker MV/MZ.

Note that the generator has been updated since the PDF guide was originally 
written.
