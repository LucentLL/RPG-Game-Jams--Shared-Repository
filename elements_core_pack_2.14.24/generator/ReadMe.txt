~~~~~~~~~~~~~~~~~~~~~~
Elements Generator 2.1
~~~~~~~~~~~~~~~~~~~~~~

To get started, simply paste all of the "assets" folders (backextra, backhair, etc...) into the
directory with "Elements Character Generator.exe". This will set up the default folders to use 
the contents of this pack.
Alternatively, you can choose an asset folder using the option at the top of the generator.

~~~~~~~~~~~~~~~~~~~~~~
Updated since the previous release:

- Added save/load characters
- New default skin tones added
- Generally runs much faster!
- Linux and OSX versions added
- Quick fix for low resolution users: you can now manually change the scale by 
editing the "Window Scale" value in the Settings.json file

~~~~~~~~~~~~~~~~~~~~~~
For further details about using the Generator, see the included PDF Guide in the "extras" folder.
