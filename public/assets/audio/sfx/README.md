# Sound effects

Drop SFX here (short `.ogg`/`.wav`). Reference via `AUDIO_SFX_BASE` from
[`src/config/assets.js`](../../../../src/config/assets.js), e.g. `AUDIO_SFX_BASE + 'hit.ogg'`.

Suggested SFX: ui-click, ui-confirm, ui-back, coin, level-up, hit, crit, block, materia-socket,
forge-clang, quest-complete.
